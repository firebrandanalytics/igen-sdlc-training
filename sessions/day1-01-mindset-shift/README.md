# Hour 1 — The Mindset Shift: From Autocomplete to Agentic Development

The opening hour. Conceptual — no Claude Code yet. We cover what an agent
actually is (planning, tool use, iteration, self-correction), the
reviewer/judge mindset, trust calibration, and where the next ten hours are
headed. The hour closes with a group discussion.

## In this folder

- `discussion-prompts.md` — prompts for the closing group discussion: where you
  sit on the autocomplete → agentic spectrum.
- `C1-course-glossary.md` — one-page glossary of terms used across the whole
  course. Keep it open from here on.

## What we're doing this hour

- Why "tab-complete" is the local maximum we're leaving behind
- Agent vs. assistant: planning, tool use, iteration, self-correction
- Engineering judgment over keystrokes — the reviewer/judge mindset
- Trust calibration: confidently-wrong output, when to verify
- Group discussion (last 10–15 min) — use `discussion-prompts.md`

## No exercise, no homework

Hour 1 is discussion only. The first hands-on work is in Hour 4.
