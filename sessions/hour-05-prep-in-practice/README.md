# Hour 5 — Prep Work in Practice: Skills, .env, and Bespoke Tooling

We build the artifacts the course has been pointing at: a CLAUDE.md that pulls
weight, a SKILL.md, .env files, and wrapper scripts as "Make Correctness Easy"
devices. Each attendee writes a SKILL.md for one of their own recurring tasks
and builds one small wrapper.

## In this folder

- `skill-and-wrapper-exercise.md` — write a SKILL.md and build one wrapper script.
- `C2-claude-md-template.md` — CLAUDE.md template + annotated example.
- `C3-skill-md-template.md` — SKILL.md template + annotated example.
- `C4-dotenv-in-practice.md` — what goes in `.env`, what doesn't.
- `C8-make-correctness-easy-checklist.md` — the patterns this hour puts to work.

## What we're doing this hour

- Writing a CLAUDE.md that actually pulls weight
- Building a SKILL.md (and when to split it into a folder)
- `.env` files: keep secrets and parameters out of the AI's working memory
- Wrapper scripts and bash functions as "Make Correctness Easy" devices
- Exercise: write a SKILL.md + build one wrapper (`skill-and-wrapper-exercise.md`)
