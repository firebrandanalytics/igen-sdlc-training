# Hour 3 — Operating the Agent: Planning, Context, and Cost

The control surface. All conceptual — hands-on follows in Hour 4. We cover
approval modes, planning mode, subagents, context management, model selection,
and how those choices compound into a sustainable (or runaway) cost.

## In this folder

  Mode, or an unsandboxed VM.
- `C6-model-selection-cheat-sheet.md` — Opus / Sonnet / Haiku trade-offs.

- Approval modes: manual → Auto Mode → unsandboxed
- Planning mode: the plan-then-execute discipline
- Subagents and agent teams: when to delegate
- Context management: CLAUDE.md, on-demand loading, `/compact`, `/clear`
- Model selection and cost optimization
- IP, provenance, and governance basics
