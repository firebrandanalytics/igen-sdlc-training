# iGEN AI Software-Development Training — Student Materials

Hands-on materials for the Firebrand-led AI software-development training for
iGEN developers — five two-hour sessions on agentic development with Claude Code.

## What's here

- **`exercises/`** — the hands-on work, one folder per session. Includes two
  starter codebases you extend during the course:
  - `session-4-5-web-app/` — the Mileage Logbook web app (FastAPI + SQLite + Jinja2)
  - `session-4-cli-utility/` — the IFTA fuel-tax CLI
  Each has its own `README.md` and `LAB-GUIDE.md`.
- **`handouts/`** — one-page reference cards: CLAUDE.md and SKILL.md templates,
  cheat sheets, checklists, and a Claude Code commands card.
- **`homework/`** — three take-home assignments: HW1 (D8) operate Claude Code on a real task; HW2 (D9) write a CLAUDE.md and SKILL.md; HW3 (D10) run the Hour 7 comprehension methodology on your own code.

## Using this repo

Clone it before Session 1. During the sessions you'll work inside the exercise
folders, directing Claude Code. Worked solutions are kept separately and are not
in this repo.
